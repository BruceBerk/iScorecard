# iScorecard
Scorecard app for tabletop baseball games

## Setup
1. Create league
2. Create teams
3. Assign teams to league
4. Load team rosters
5. Add player ratings

## Play
1. Create a game matchup
2. Score the game
3. Save the game

## Stats
1. View/Export standings
2. View/Export league leaders
3. View/Export team stats
4. Encyclopedia
